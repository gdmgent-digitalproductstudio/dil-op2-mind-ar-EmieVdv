Hoe targets verkregen?

De kaarten zijn online gevonden via bestaande NBA- en WNBA trading card visuals.
De afbeeldingen werden gedownload van het internet en gecomprimeerd zodat hiervan makkelijk markers gemaakt konden worden.


Testmethode

Tijdens het testen werden de targets op twee manieren gebruikt:

- via de Virtual Camera van OBS Studio
- via de webcam, waarbij de targets getoond werden op een smartphone
